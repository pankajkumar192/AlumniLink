## 🐛 TROUBLESHOOTING

### MongoDB Connection Error
**Solution**: Ensure MongoDB is running or update MONGODB_URI in backend/.env

### Port Already in Use
**Solution**: Change PORT in backend/.env or kill process using that port

### npm install fails
**Solution**: Delete node_modules and package-lock.json, then run npm install again

### Can't see changes
**Solution**: Hot reload is enabled - file changes should auto-update. Refresh browser if needed
